# REX: France Télévisions, quand API Platform tombe à PIC

Hello la communauté,

Prochain rendez-vous le jeudi 20 Avril au CEFIM où nous accueillerons **Albin Kester** des tilleuls.coop pour vous parler de "**REX: France Télévisions, quand API Platform tombe à PIC**".

Un membre de l’équipe de la Plateforme d’Information Commune (PIC) de la Direction du Numérique de France Télévisions vous présentent un retour d’expérience sur l’utilisation de API Platform. Je vais présenter l’architecture et l’utilisation des différentes API pour supporter la charge des sites d’actualités de France 3 Régions, le Portail des Outre-mer La 1ère et Franceinfo.
