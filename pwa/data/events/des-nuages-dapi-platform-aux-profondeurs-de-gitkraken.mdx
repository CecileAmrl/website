# Des nuages d'API Platform aux profondeurs de GitKraken !

Au menu de la soirée :

- un apéritif avec l'actu news de l'antenne AFUP ~5min
- une entrée dans API Platform avec Antoine Obligis ~20min + questions
- un plat copieux avec GitKraken avec Florian Montalbano et Sébastien Courjean ~35min + questions

L'API vous intéresse ? Vous souhaitez créer un projet en API-first, mais vous ne savez pas encore quel framework Back-end choisir ?
Nous vous proposons de découvrir API-platform ! Simple et puissant, ce framework « Français » basé sur symfony vous fera gagner du temps de développement !
Au sommaire, présentation des fonctionnalités et comment le prendre en main rapidement.

Vous souhaitez utiliser Git dans votre projet mais vous n'avez pas de place pour une cheat sheet supplémentaire sur votre bureau ?
Nous vous proposons d'aller à la découverte de GitKraken, une interface graphique très puissante pour naviguer sur les océans des repos Git.
Les arbres Git seront désormais clairs pour toute l'équipe, GitFlow sera votre allié du quotidien et la résolution de conflits un jeu d'enfant.
