# Créer une web app & une app mobile en React Native grâce à API Platform

Venez créer une web app et une app mobile en React Native grâce au framework API Platform

Vous découvrirez comment utiliser API Platform pour créer rapidement des applications riches utilisant ReactJS pour leur couche de présentation.

Linkvalue reçoit Grégoire de chez Les-Tilleuls.coop, qui vous propose de créer pas à pas une API web 100% fonctionnelle grâce à API Platform, Symfony et Doctrine.
Cette API supportera la validation, la pagination, les filtres, l’imbrication de ressources, sera documentée via Swagger et Hydra et disposera d’une interface graphique orientée développeur.

Nous découvrirons ensuite les tout nouveaux outils front-end du projet API Platform :
Un système d’administration complet (à la Sonata), construit automatiquement en découvrant l’API et bénéficiant d’une interface moderne (Material Design) basée sur React et Redux
Un générateur de code permettant de créer des interfaces ReactJS en CRUD, à la manière des outils de génération de code fournis par Symfony (mais côté client).

La soirée se terminera par un apéro - questions !
